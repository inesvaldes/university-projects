Main Menu

The main menu is the starting point of the OpenWorld Agency application. From here, you can choose whether to begin your booking with theme parks or accommodations. This panel also highlights the theme park that is currently on special offer.

Elements on the screen

  • Logo: Displays the OpenWorld Agency brand. It is purely decorative and has no direct action.
  • Application title: Shows the name of the application, for example “OpenWorld Agency”.
  • Special offer area: A highlighted section showing that one park is on offer. All tickets and accommodations associated with this park have a 15% discount on the final price.
  • Featured park name and image: Indicates which theme park is currently on offer, including an image and sometimes a short description.
  • Theme parks section image and button: An image representing theme parks and a button that opens the Theme Parks panel.
  • Accommodations section image and button: An image representing accommodations and a button that opens the Accommodations panel.

How the special offer works

Each time the application is launched, one theme park is selected randomly as the offer park. This park remains on offer throughout the entire session. All tickets and all accommodations associated with this park receive a 15% discount on their final price. The discount will be shown explicitly in the reservation summary if the user has booked items for that park.

How to use the Main Menu

  1. Read the information about the special offer and the featured park.
  2. Decide whether you want to start with Theme Parks (tickets) or Accommodations.
  3. Click the corresponding button:
    • Theme Parks: Opens the Theme Parks panel.
    • Accommodations: Opens the Accommodations panel.
  4. You can always come back to the Main Menu from other panels using the Back button, although doing so may reset your current reservation.
