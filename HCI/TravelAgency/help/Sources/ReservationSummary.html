Reservation Summary panel

The Reservation Summary panel displays all the details of the booking in a clear, formatted way. This is the last step before confirming payment and saving the booking to a file.

Summary content

The summary follows a structure similar to the one described in the project statement. It normally includes:

  • Company name and booking header.
  • Booking date and theme park name.
  • Customer ID and customer name.
  • Tickets section:
    • Start date
    • Number of days
    • Number of adults and children
  • Accommodation section:
    • Accommodation code, type, category, and name
    • Start date and number of nights
    • Number of people
  • Booking amount section:
    • Tickets total
    • Accommodation total
    • Offer discount (if applicable)
    • Final total amount
  • Possible comments or observations added in the Reservation panel.

Elements on the screen

  • Summary text area: A scrollable area containing the formatted booking summary.
  • Confirm button: Confirms the booking, creates the text file with all the summary information, and moves to the Mini Game panel.
  • Cancel button: Cancels the entire booking process and returns to the Main Menu, leaving the application ready for the next customer.

File creation

When you click Confirm, the application saves the reservation information into a text file. The file name includes the customer’s ID number and the date and time of the booking. This file acts as proof of payment and contains all the details shown in the summary, and, if applicable, a line indicating that a 100€ voucher was obtained in the mini game.

How to use this panel

  1. Carefully read all the information in the summary text area.
  2. Check:
    • Dates and number of days/nights.
    • Number of people and rooms.
    • Park and accommodation details.
    • Discount amount and final total.
  3. If something is wrong, you can go back to previous panels (Reservation or Customer Information) and correct it.
  4. If everything is correct:
    • Click Confirm to save the booking and move to the Mini Game.
    • Or click Cancel to discard everything and return to the Main Menu.
