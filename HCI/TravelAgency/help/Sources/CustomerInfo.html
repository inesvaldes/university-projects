Customer Information panel

In the Customer Information panel you enter the personal data required to formalise the booking. All fields are mandatory and must follow specific validation rules.

Elements on the screen

  • First name field: The customer’s first name. Must contain only letters.
  • Last name field: The customer’s last name(s). Must contain only letters.
  • ID number field: The customer’s identification document number. It must be composed of exactly 8 digits followed by a single letter (e.g. 12345678A).
  • Finish button: Validates the information and, if it is correct, registers the customer data and moves to the Reservation Summary panel.
  • Back button (if present): Returns to the Reservation panel to review or change the booking.

Validation rules

  • All fields are required. You cannot continue with any empty fields.
  • The first name and last name fields should contain letters only (no numbers).
  • The ID number must follow this format:
    • 8 digits (0–9)
    • followed by exactly one letter (A–Z)

How to proceed

  1. Fill in all three fields: first name, last name, and ID number.
  2. Check that the ID number follows the required format (8 digits + 1 letter).
  3. Click Finish.
  4. If there are errors, the application will notify you and you must correct them before you can continue.
  5. If everything is correct, you will be moved to the Reservation Summary panel.
