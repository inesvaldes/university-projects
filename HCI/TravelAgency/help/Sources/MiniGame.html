Mini Game panel

After confirming the reservation, the customer is rewarded with a small mini game. The game offers the possibility of winning a 100€ voucher for the next booking.

Game description

The mini game consists of a board with five hidden cells. Only one of these cells contains a 100€ gift voucher; the others are empty or contain a losing symbol. The customer can uncover only one cell by clicking on it once.

Elements on the screen

  • Instructions label: A short explanation of the game rules for the user.
  • Game cells: Five clickable cells, initially displayed with a question mark (or similar icon) to indicate that they are hidden.
  • Result label: A label informing the user whether they have won the 100€ voucher or not.
  • Revealed voucher location: If the player loses, the game reveals which cell contained the voucher.
  • Close button: Closes the mini game and returns to the Main Menu, leaving the application ready for the next customer.

How to play

  1. Read the instructions at the top of the panel.
  2. Click on one of the five cells. Only the first click is valid; afterwards, the other cells are disabled.
  3. The selected cell is revealed:
    • If it shows a voucher symbol (for example, a “%” sign), the customer wins a 100€ gift voucher for the next booking.
    • If it shows a losing symbol (for example, a cross), the customer does not win the voucher.

  4. In case of losing, the cell that contained the voucher is revealed so that the user can see where it was.
  5. The result label at the bottom displays a message summarising the outcome of the game.

Effect on the booking file

If the customer wins the 100€ voucher, an additional line is added to the booking file, after the “Offer discount” line and before “Total amount”: 

Obtained a 100€ gift voucher for your next booking

The reservation summary itself does not have to be displayed again; it is enough that the gift is recorded in the file.

Finishing the game

When the game is finished, the customer can click the Close button. This returns the application to the Main Menu, with the random offer park preserved but all booking data cleared, so the system is ready for the next client.
