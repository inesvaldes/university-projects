Theme Parks panel

In the Theme Parks panel you can select a theme park, view its detailed information, and book tickets for adults and children for a specific date and number of days. You can also jump to the accommodations panel for the selected park.

Elements on the screen

  • Theme park image: Displays a photo of the currently selected park. The image is automatically resized to fit the component.
  • Offer indicator: A text area or label indicating whether the selected park is the one on offer (15% discount) or not.
  • Park selection combo box: A drop-down list containing all available theme parks. When you select a park, the image and the information area are updated.
  • Park information area: A scrollable text area showing details of the park: name, country, location, description, and ticket prices for adults and children.
  • Entry date field: A field or spinner to select the start date of the tickets.
  • Number of days: A field or spinner indicating how many days of tickets you want.
  • Number of adult tickets: A field or spinner to select how many adult tickets to book.
  • Number of child tickets: A field or spinner to select how many child tickets to book.
  • Recommended accommodation button: Reserves the selected park (if valid data is entered) and moves to the Accommodations panel, showing only accommodations associated with this park.
  • Book button: Adds the selected park tickets to the reservation and moves directly to the Reservation panel.
  • Back button: Returns to the Main Menu and resets all current reservation data, while keeping the same offer park for the session.

How to book park tickets

  1. Select a park from the park combo box.
  2. Check the information area to confirm that this is the park you are interested in.
  3. Select:
    • Entry date
    • Number of days
    • Number of adult tickets
    • Number of child tickets
  4. If the park is on offer, remember that the discount will be applied later in the reservation summary.
  5. Click:
    • Recommended accommodation if you want to continue by choosing an accommodation for this same park.
    • Book if you want to go directly to the reservation list.

Notes

  • Tickets must always be associated with exactly one theme park.
  • You can modify or remove this booking later from the Reservation panel.
